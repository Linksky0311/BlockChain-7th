## 구현된 기능

### 기본 기능 (필수)
| 함수 | 설명 |
|------|------|
| `stake()` | ETH를 컨트랙트에 예치 (payable) |
| `withdraw(uint256 amount)` | 스테이킹된 ETH 인출 |
| `getBalance()` | 내 스테이킹 잔액 조회 |

### 추가 기능 (보너스)
| 기능 | 설명 |
|------|------|
| **보상 계산** | 연 10% APR 기반 시간 비례 보상 자동 계산 |
| **Time-lock** | 7일 락업 기간 (LOCK_PERIOD = 7 days) |
| **긴급 인출** | `emergencyWithdraw()` - 10% 패널티 적용 즉시 인출 |
| `calculateReward()` | 미청구 보상 조회 |
| `getStakeInfo()` | 스테이킹 상세 정보 (금액, 시작시간, 남은 락업시간 등) |
| `getContractBalance()` | 컨트랙트 전체 잔액 조회 |

---

## 컨트랙트 주요 설계

```
- LOCK_PERIOD: 7일 (604800초)
- APR: 10% 연이율
- EMERGENCY_PENALTY: 10% (긴급 인출 시 패널티)
- 보상 계산식: (원금 × APR × 경과시간) / (100 × 1년)
- 보안 패턴: Checks-Effects-Interactions 적용
```
